# Sistem Pengelolaan Surat Peringatan Mahasiswa

Sistem ini dikembangkan untuk **memfasilitasi proses administrasi terkait surat peringatan mahasiswa** di lingkungan perguruan tinggi. Dengan memanfaatkan aplikasi ini, staf akademik dan administrasi dapat membuat, menyimpan, serta memantau surat peringatan secara digital, sehingga prosedur pengelolaan menjadi lebih efisien, transparan, dan terdokumentasi dengan baik.

---

## Tujuan dan Fungsi

Sistem ini memiliki beberapa tujuan utama:  

- **Menyederhanakan pembuatan surat peringatan** bagi mahasiswa yang melanggar peraturan institusi.  
- **Memelihara catatan administratif** secara terstruktur dan mudah diakses.  
- **Mendukung pengambilan keputusan berbasis data** melalui laporan dan statistik pelanggaran.  
- **Meningkatkan akurasi dan efisiensi administrasi** melalui fitur notifikasi dan ekspor data.  

---

## Fitur Utama

1. **Pembuatan Surat Peringatan**
   - Mendukung berbagai tingkatan surat peringatan (SP1, SP2, SP3).  
   - Memuat data mahasiswa secara lengkap dan alasan pelanggaran.  

2. **Manajemen Data Mahasiswa**
   - Penyimpanan data yang terorganisir dan mudah dicari.  
   - Filter berdasarkan tingkat surat peringatan, program studi, atau semester.  

3. **Laporan**
   - Laporan bulanan, semesteran, atau tahunan terkait surat peringatan.    

4. **Notifikasi dan Pengingat**
   - Pengingat otomatis untuk tindak lanjut surat peringatan.  

5. **Ekspor dan Dokumentasi**
   - Ekspor data ke format PDF untuk keperluan administrasi dan arsip.
