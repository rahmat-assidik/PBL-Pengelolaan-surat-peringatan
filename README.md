# Sistem Pengelolaan Surat Peringatan Mahasiswa

Sistem ini adalah aplikasi digital untuk **mempermudah pengelolaan surat peringatan mahasiswa** di lingkungan perguruan tinggi. Dengan sistem ini, staf akademik dan administrasi dapat membuat, menyimpan, dan memantau surat peringatan secara efisien, transparan, dan terdokumentasi dengan baik.

---

## Fitur Utama

- **Pembuatan Surat Peringatan**
  - Mendukung SP1, SP2, SP3.
  - Menyertakan data mahasiswa dan alasan pelanggaran.  

- **Manajemen Data Mahasiswa**
  - Penyimpanan data terstruktur.
  - Filter berdasarkan tingkat surat peringatan, program studi, atau semester.  

- **Laporan**
  - Laporan bulanan, dan semesteran. 

- **Ekspor dan Dokumentasi**
  - Ekspor data ke PDF untuk administrasi dan arsip.  

---

## Tujuan

- Mempermudah pembuatan surat peringatan mahasiswa.  
- Menyimpan data administratif secara terstruktur.  
- Mendukung pengambilan keputusan berbasis data.  
- Meningkatkan efisiensi administrasi melalui notifikasi dan ekspor data.
